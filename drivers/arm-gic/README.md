# Arm Generic Interrupt Controller driver

This crate provides a specialized Rust driver for the Arm Generic Interrupt Controller version 3(GICv3).Currently, the implementation focuses on the AArch64 architecture. It serves as a core hardware abstraction layer (HAL) component, ensuring efficient interrupt delivery and management within the x-kernel environment.

This driver is maintained as an integral part of the KylinSoft x-kernel ecosystem.
